# 快速开始指南

## 安装

### 1. 找到 WebUI Forge 的 extensions 目录

```
stable-diffusion-webui-forge/extensions/
```

### 2. 克隆或复制本项目

**方法 A：使用 Git**
```bash
cd stable-diffusion-webui-forge/extensions/
git clone <repository-url> sd-webui-watermark-adder
```

**方法 B：手动复制**

将整个项目文件夹复制到 `extensions` 目录下

### 3. 重启 WebUI Forge

关闭并重新启动 WebUI Forge，在顶部标签栏找到 **Watermark Adder** 标签页。

## 界面布局

界面分为三栏：

| 左栏：水印库 | 中栏：编辑区 | 右栏：预览与保存 |
|:---|:---|:---|
| 图片/文字水印管理 | 图像编辑、水印放置 | 生成预览、保存图片 |
| 上传、删除、选择 | 缩放、平移、快捷键 | 普通保存 / 可解压包 |

## 使用教程

### 步骤 1：上传图片

- 在编辑区拖放或点击上传图片
- 或点击 **获取上次生成的图片**（自动扫描 `outputs/` 文件夹获取最新图片，排除水印输出目录）

### 步骤 2：创建或选择水印

**创建文字水印：**
1. 切换到"文字水印"标签页
2. 输入水印文字（如 "© 2024"）
3. 调整字体大小、颜色
4. 输入水印名称并点击"保存"

**上传图片水印：**
1. 切换到"图片水印"标签页
2. 拖放或点击上传水印图片（建议 PNG 透明背景）
3. 上传后自动保存到水印库

### 步骤 3：添加水印到图片

1. 在水印库中点击选择一个水印
2. 鼠标移动到编辑区，水印跟随鼠标实时预览
3. 使用快捷键调整：
   - **Ctrl + 滚轮**：调整大小（±20px）
   - **Shift + 滚轮**：调整角度
   - **Alt + 滚轮**：调整透明度
4. **单击左键** 放置水印
5. 可以重复添加多个水印

### 步骤 4：图像浏览

在编辑区：
- **滚轮**：缩放图片
- **长按左键拖拽**：平移图片
- **双击**：重置缩放和平移

### 步骤 5：生成和保存

1. 点击 **生成水印图片**（默认自动保存到 `outputs/watermarked/`）
2. 在右侧预览效果
3. 选择保存方式：
   - **保存图片**：保存为标准 PNG
   - **保存可解压包**：保存为特殊格式

## 可解压图片包

这是一个特殊功能，生成的文件：
- 看起来是普通的 PNG 图片（带水印版本）
- 将文件后缀 `.png` 改为 `.zip`，用解压软件打开即可得到无水印原图

**使用方法：**
```bash
# Windows
ren extractable_1234567890.png extractable_1234567890.zip

# Linux/Mac
mv extractable_1234567890.png extractable_1234567890.zip

# 解压
unzip extractable_1234567890.zip
# 得到 original_image.png
```

## 常见问题

### Q: 为什么看不到 Watermark Adder 标签页？
确保项目放在 `extensions` 目录下，已经重启 WebUI Forge，检查控制台是否有报错。

### Q: 大图加载显示黑屏？
扩展会自动重试等待图片解码完成（最多约 9 秒）。如果图片特别大，请耐心等待或缩小图片后上传。

### Q: 保存的图片在哪里？
默认保存在 WebUI 根目录下的 `outputs/watermarked/`。

### Q: 可解压图片包无法解压？
确保文件后缀已改为 `.zip`，使用支持 ZIP 格式的解压软件（WinRAR、7-Zip、unzip 等）。

### Q: 水印文字显示为方块？
系统缺少对应字体，会自动使用默认字体。可以使用图片水印代替。
